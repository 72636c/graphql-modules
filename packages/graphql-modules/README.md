**GraphQL Modules** is a toolset of libraries and guidelines dedicated to **create reusable, maintainable, testable and extendable modules** out of your GraphQL server.

You should use [`@graphql-modules/core`](https://www.npmjs.com/package/@graphql-modules/core) npm package instead.
See [docs](https://graphql-modules.com) and [repository](https://github.com/Urigo/graphql-modules) about details.
